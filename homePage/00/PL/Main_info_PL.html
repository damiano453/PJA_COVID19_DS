Jesteśmy studentami kierunku Big Data na Polsko Japońskiej Akademii Technik Komuputerowych w Warszawie.

W ramach pracy zaliczeniowej stworzyliśmy serwis obrazujący rozprzestrzenianie się wirusa COVID-19 na całym świecie oraz śmiertelność.

Strona przedstawia analizę danych statystycznych przeliczanych w sposób umożliwiający obiektywne ich porównanie pomiędzy krajami.

Dane zostały zwizualizowane na mapie świata oraz przy użyciu innych technik prezentacji np. map wygrzewanie etc.

Istotnym elementem są przygotowane tutoriale opisujące skrypty generujące wyniki, stworzone w języku R. Dodatkowo stworzone zostało Rest API umożliwiające tworzenie zapytania do bazy danych przechowującej przetworzone oraz surowe dane. Rest API jest udosępnione w sekcji Pobierz.
