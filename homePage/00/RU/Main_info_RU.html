Мы являемся студентами Big Data Польско-Японской Академии Компьютерных Наук в Варшаве.

Данный сервис создан в качестве дипломного проекта, который отображает распространение вируса COVID-19 по всему миру.

Вебсайт показывает статистические данные, которые могут быть использованы в целях объективной оценки ситуации в разных странах.

Данные визуализированны с помощью карты мира, а также различных техник презентации, таких как тепловая карта.

В дополнение, было приготовлено практическое руководство, которое объясняет процесс создания скриптов в языке R использованных для презентации данных. Дополнительно, Rest API были созданы для готовой базы данных и размещены в секции Скачать, которые пользователь может использовать в своих целях.
